# PDR 001: Product scope

- Версия: draft
- Дата: 2026-04-13

## Цель продукта

Создать цифровую платформу для охранной компании, которая покрывает:

- презентацию компании и сбор лидов через публичный сайт;
- управление организациями, сотрудниками и доступами;
- рабочие кабинеты для разных ролей;
- мобильный доступ к операционным сценариям;
- внутренний чат и оперативные уведомления.

## MVP

В первый контур MVP входят:

1. Публичный сайт компании.
2. Авторизация и восстановление доступа.
3. Кабинет администратора платформы.
4. Кабинет руководителя организации.
5. Кабинет сотрудника организации.
6. Создание сотрудников руководителем.
7. Назначение и отзыв доступа.
8. Базовый чат по организациям и сотрудникам.
9. GitHub Actions pipeline и Docker deploy на сервер.

## Вне MVP

- сложная биллинговая модель;
- white-labeling;
- аналитика enterprise-уровня;
- offline-first синхронизация;
- продвинутый AI layer.
